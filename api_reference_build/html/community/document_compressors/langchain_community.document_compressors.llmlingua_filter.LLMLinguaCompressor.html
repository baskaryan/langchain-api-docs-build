extract_ref_id_tuples_and_clean(contents: List[str]) ‚Üí List[Tuple[str, int]][source]#

Extracts reference IDs from the contents and cleans up the ref tags.

This function processes a list of strings, searching for reference ID tags at the beginning and end of each string. When a ref tag is found, it is removed from the string, and its ID is recorded. If no ref ID is found, a generic ID of ‚Äú-1‚Äù is assigned.

The search for ref tags is performed only at the beginning and end of the string, with the assumption that there will be at most one ref ID per string. Malformed ref tags are handled gracefully.

Parameters:

contents (List[str]) ‚Äì A list of contents to be processed.

Returns:

The cleaned string and the associated ref ID.

Return type:

List[Tuple[str, int]]

Examples

>>> strings_list = [
        '<#ref0#> Example content <#ref0#>',
        'Content with no ref ID.'
    ]
>>> extract_ref_id_tuples_and_clean(strings_list)
[('Example content', 0), ('Content with no ref ID.', -1)]
